## Key Features

<CardGrid stagger>
	<Card title="Isolated Class Loading" icon="seti:java">
		Plugins are loaded in complete isolation using a custom `StreamClassLoader`, preventing dependency conflicts.
	</Card>
	<Card title="Type-Safe Contracts" icon="approve-check-circle">
		Define clear interfaces (contracts) that plugins must implement for seamless integration.
	</Card>
	<Card title="JSON Metadata" icon="document">
		Simple `plugin.json` metadata for discovery, validation, and identification.
	</Card>
	<Card title="Lifecycle Management" icon="rocket">
		Easily manage the lifecycle of your plugins with the `PluginManager`.
	</Card>
</CardGrid>
